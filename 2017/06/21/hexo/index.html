Hexo 安装使用

  1. 先安装nodejs
  2. 安装hexo
    npm install -g hexo
  3. 初始化然后，执行init命令初始化hexo
    init 会使用 git 所以要把git加入到环境变量
    hexo init
    安装完成

常用命令

  • 生成静态页面
    hexo generate（hexo g也可以）
  • 启动本地服务，进行文章预览调试
    hexo server
  • 生成新文件
    hexo n 文件名
  • 更新hexo到最新版
    npm update hexo -g
